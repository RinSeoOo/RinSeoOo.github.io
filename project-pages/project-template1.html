Infra&Web 취약점진단
SK쉴더스 루키즈 과정에서 팀 단위로 AWS 환경에 웹 애플리케이션을 구축하고, 이를 대상으로 KISA 기반시설 기술적 취약점 분석 평가 상세 가이드와 SK쉴더스 내부 가이드 기준으로 취약점 진단을 수행했습니다. 인증·세션, 입력 검증, 서버 설정 등 주요 영역을 중심으로 식별된 취약점에 대해 이행 조치를 적용했으며, 최종적으로 보안 설정을 강화한 개선 버전을 도출했습니다.
Project Details / Background
역할 및 세부 수행 내역
- 주어진 웹 사이트에 대해 취약점 진단을 하기 위해 세부 기능을 구현했다.
- KISA 및 SK쉴더스 내부 가이드를 기반으로 웹 취약점 진단을 수행하고, 식별된 취약점에 대한 개선 조치를 적용했다.
- 취약점진단 수행 후, 수동진단과 자동진단 결과물을 비교하기 위해 자동화 스크립트 제작을 맡았다.
- 진단 결과와 개선 사항을 기반으로 분석 보고서를 작성하여 프로젝트 산출물의 완성도를 높였다.
수행 작업
- AWS 기반의 3-Tier 아키텍처를 구성하여 취약점 진단 환경을 구축했다.
- Web은 Ubuntu 기반 Apache, WAS는 Debian 기반 Tomcat, DBMS는 Windows 환경의 Oracle로 각각 분리 구성하여 실제 서비스와 유사한 구조에서 진단을 수행했다.
결과
- 취약점에 대한 이행조치 이후 평가등급(TCSEC) 기준으로 50.2%였던 보안수준 점수가 98.0%로 평균 47.8%p 증가했다.
